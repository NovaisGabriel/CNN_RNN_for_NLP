RNN - LSTM x GRU

O objetivo deste cdigo apresentar um exemplo simples de aplicao de redes neurais recorrentes (RNN) para a anlise de textos (NLP). Neste caso, vamos aplicar deep learning para obter a partir do comentrio realizado a cerca de um deteminado filme se este comentrio foi ou no positivo. Os dados foram retirados do IMDB, e cujo link segue abaixo:

Dados de comentrios sobre IMDB

Vale observar que como so dados utilizados justamente para testes ento eles costumam vir j pr-processados de maneira que no necessrio fazer limpezas nos comentrios, excluindo por exmeplo caracteres errados ou smbolos. O que ser feito justamente o procedimento de tokenizao para depois aplicar os mtodos de LSTM e GRU.
